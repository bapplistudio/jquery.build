jquery.build simple example

The example

The code

You can tell jquery to call events for new dynamically added elements with .on(), but what about other jquery initialisations, such as jquery-ui widgets building ?

jquery.build helps you to do that.

Typical initialization code, without build :
The helloworld event will ony be set for the initial button, not for the dynamically added buttons : 

$("document").ready(function() {
	// This code is executed only when you load the page, and not for dynamically added elements

	$(".helloworld").click(function() {
		alert("Hello World");
	});

	$("#add").click(function() {
		$("#example").append($("<button>").addClass("helloworld").html("Hello world"));
	});
});

Now use build and keep dynamics :
The helloworld event is set for the initial button, and for the dynamically added buttons : 

$("document").ready(function() {
	$("body").build(function() {
		// This code will be executed each time you call $.build()

		this.in(".helloworld").click(function() {
			alert("Hello World");
		});

		this.in("#add").click(function() {
			$("#example").append($("<button>").addClass("helloworld").html("Hello world").build());
		});

	});
});

To make this work the best :

  • You should make all your jquery initializations into the $("body").build(function() { (...) }); section
  • You should call build() for each dynamically elements, including ajax calls
