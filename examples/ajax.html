jquery.build ajax example

The example

(the container for ajax result)

The code

You can tell jquery to call events for new dynamically added elements with .on(), but what about other jquery initialisations, such as jquery-ui widgets building ?

jquery.build helps you to do that.

The code of the example : 

$("document").ready(function() {
	$("body").build(function() {
		// This code will be executed each time you call $.build()

		this.in(".helloworld").click(function() {
			alert("Hello World");
		});

		this.in("#add").click(function() {
			$.ajax({
				url: "http://saf.re/github/jquery.build/examples/_ajax.html",
				error: function() {
					alert("error loading ajax data");
				},
				success: function(data) {
					$("#target").html(data).build();
				}
			});
		});

	});
});
